#DASHBOARD DE VENDAS – ANÁLISE DE DADOS NO MICROSOFT EXCEL

![Dashboard de Vendas](https://github.com/user-attachments/assets/2b6dfdea-43f0-48e4-b133-5a7e367f4128)


DESCRIÇÃO DO PROJETO
- Criação de um painel dinâmico e intuitivo para análise de desempenho de vendas, permitindo a visualização de tendências, metas, evolução de receitas e performance de vendedores e produtos.

FERRAMENTAS E TÉCNICAS UTILIZADAS
- Microsoft Excel (Tratamento de Dados, Fórmulas Avançadas, Tabelas Dinâmicas)
- Power Query (Limpeza e Transformação de Dados)
- Gráficos Dinâmicos (Visualização de Tendências e Comparações)
- Segmentação de Dados (Interatividade no Dashboard)
- Cálculo de KPIs (Receita, Metas, Ranking de Vendedores)

PRINCIPAIS ANÁLISES REALIZADAS
- Meta vs Receita: Comparação do desempenho mensal com as metas estabelecidas.
- Receita por Produto: Identificação dos produtos mais vendidos e suas variações ao longo do tempo.
- Análise de Vendedores: Ranking de vendedores baseado em faturamento total.
- Análise por Ano: Evolução do faturamento anual com segmentação interativa.
- Correlação entre Vendedores e Produtos: Relação entre produtos vendidos e desempenho individual dos vendedores.
